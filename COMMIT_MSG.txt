feat(home): Implement dynamic home dashboard with real-time stats and tasks

This commit transforms the home dashboard into a reactive, data-driven experience by integrating real-time Firestore statistics and task tracking. It also modularizes the UI by extracting complex sections into reusable widgets for better maintainability.

feat(home):
- Implement HomeViewModel with providers for real-time dashboard statistics, upcoming tasks, and task history.
- Refactor HomeView and HomeSliverGrid into ConsumerWidgets for reactive state management with Riverpod.
- Extract "Upcoming Tasks" and "History" logic into HomeUpcomingTasksSection and HomeHistorySection widgets.
- Add HomeUpcomingEmpty and HomeHistoryEmpty widgets to handle empty states gracefully.
- Integrate LoadingIndicator and CustomErrorWidget for robust asynchronous data handling.

feat(tasks):
- Update TaskService to strictly filter tasks by userId (creator or assignee) in watchTasksForUser and watchTasksForUserByDate.
- Enhance task categorization logic for "Pending Tasks," "Current Tasks," and "Reminders" on the dashboard.
- Refactor TaskItemCard and history displays to utilize real-time task data from Firestore.
